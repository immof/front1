배경과 배경 이미지

background-color : 요소의 배경색을 지정하는 속성

선택자{background-color : 색상명 | #16진수 | rgb | rhba}

background-clip : 배경 범위를 조절

선택자{background-clip : border-box | padding-box | content-box}

글씨

backround-image : 요소의 배경을 이미지로 지정

선택자{background-image : url(이미지경로)}

background-repeat : 배경이미지 반복을 조절하는 속성}

선택자{background-repeat : repeat | repeat-x | repeat-y | no-repeat

background-position : 배경이미지 위치 조정

background-size : 배경이미지 크기 조절

선택자{background-size:auto|contain|cover|크기지정}

background-origin : 배경이미지 배치기준

선택자 {background-origin:border-box | padding-box | content-box} -> background-clip의 색상적용범위의 이미지 버전

background-attachment : 웹페이지 상하 이동시 배경이미지 고정 여부 설정

선택자{background-attachment: scroll | fixed }
